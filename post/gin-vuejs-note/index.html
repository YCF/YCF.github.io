Gin + Vuejs 做小工具小记

2019-08-04 Golang ， Vuejs

通过实际需求驱动，用 Gin (golang web 框架)写了一个没到一千行代码的伪Restful Api后端，然后配合 vue.js 做了个前后端分离的小工具。收获也不少：

  1. Gin 连接 SQLte3，CRUD 返回 json；
  2. 实现简单模糊搜索的关键字拆分和SQL查询拼接。对多关键字空格间隔进行sql ike 和关键字以-开头实现 sql not like 查询的sql查询语句拼接；
  3. SQL 查询结果到处为 Excel；
  4. 后端分页、排序；
  5. 初步入门Vue.js，纯CDN引用，没用webpack等打包工具。
  6. 使用 Buefy 这个 vuejs的组件库（使用 Bluma CSS库，比较好用），大概知道了 vue.js 组件间通信和一些事件处理方式。
